# Privacy Policy Draft

`In Salah` uses the following data to provide core functionality:

- Location: used to calculate prayer times for the user's area.
- Account information: email and authentication details for sign-in and profile sync.
- Optional profile image: used only for the user's profile.
- Prayer setup preferences: calculation method, madhab, iqama offsets, reminder settings, and interface preferences.

The app does not sell personal data.

Location is used only to calculate prayer times and related reminders.

Reminder settings are stored so the app can schedule local notifications on the device.

If the user signs in, profile data may be synced with the backend linked to the app.

Users can request account deletion by contacting the support channel listed in the app or in the Play listing.

This draft should be published at a public HTTPS URL before Google Play submission.
